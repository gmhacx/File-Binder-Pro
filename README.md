
# File-Binder-Pro
A Simple Proof of Concept File Binder that can combine two different files into another file which silently extracts them when opened. Written in C#.
# Usage
- Build from source or download install.zip
- Run FileBinderPro.exe
- Go to Files Tab and Right Click to Bring up the Context Menu
- Click Add and Add any Files Required
- Note: Other Tabs Can be Used to Edit Properties of Resultant File
- Go to Build Tab to Generate File
# Build Instructions
**Requirements:**
- Visual Studio with C#
**Instructions:**
- Open FileBinderPro.sln
- Click Run at Top of Visual Studio

*Note: Can be Compiled for Other Platforms with Slight Tweaks*
